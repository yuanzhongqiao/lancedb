<p dir="auto" _msttexthash="502063679" _msthash="269">LanceDB 是一个使用持久存储构建的用于向量搜索的开源数据库，可大大简化嵌入的检索、筛选和管理。</p>
<p dir="auto" _msttexthash="58891456" _msthash="270">LanceDB 的主要功能包括：</p>
<ul dir="auto">
<li>
<p dir="auto" _msttexthash="105025778" _msthash="271">生产规模的向量搜索，无需管理服务器。</p>
</li>
<li>
<p dir="auto" _msttexthash="298756614" _msthash="272">存储、查询和筛选矢量、元数据和多模态数据（文本、图像、视频、点云等）。</p>
</li>
<li>
<p dir="auto" _msttexthash="72831226" _msthash="273">支持向量相似性搜索、全文搜索和 SQL。</p>
</li>
<li>
<p dir="auto" _msttexthash="41420951" _msthash="274">本机 Python 和 Javascript/Typescript 支持。</p>
</li>
<li>
<p dir="auto" _msttexthash="220752064" _msthash="275">零拷贝、自动版本控制，无需额外的基础设施即可管理数据版本。</p>
</li>
<li>
<p dir="auto" _msttexthash="67996383" _msthash="276">构建向量索引 （*） 的 GPU 支持。</p>
</li>
<li>
<p dir="auto" _msttexthash="549034785" _msthash="277">与 <a href="https://python.langchain.com/docs/integrations/vectorstores/lancedb/" rel="nofollow" _istranslated="1">LangChain 🦜️🔗</a> 、 <a href="https://gpt-index.readthedocs.io/en/latest/examples/vector_stores/LanceDBIndexDemo.html" rel="nofollow" _istranslated="1">LlamaIndex 🦙</a> 、 Apache-Arrow、Pandas、Polars、DuckDB 等的生态系统集成正在进行中。</p>
</li>
</ul>
<p dir="auto" _msttexthash="693791540" _msthash="278">LanceDB 的核心是用 Rust 🦀 编写的，并使用 <a href="https://github.com/lancedb/lance" _istranslated="1">Lance 构建，Lance</a> 是一种开源列式格式，专为高性能 ML 工作负载而设计。</p>
